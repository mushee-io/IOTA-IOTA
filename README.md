# IOTA Builder Passport

Premium demo concept for an IOTA-native builder trust product.

## Routes

- `/` — public landing page with a black-and-white prelanding feel and full startup-style homepage sections
- `/dashboard` — white-and-pink product dashboard with interactive demo controls

## What this demo shows

- builder passport identity flow
- verifiable credential issuance
- notarized proof bundle workflow
- grant committee review desk
- grant ops and issuer controls
- hackathon and workshop flows
- analytics, reputation, and mainnet access gating

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Notes

This is a polished prototype intended for grant applications, product storytelling, and investor-style demos.
